# 📚 Ingeniería de Software 3 — UNLP

> Material de estudio de la materia **Ingeniería de Software 3** de la Facultad de Informática, Universidad Nacional de La Plata (UNLP).

---

## 📌 Descripción

Este repositorio reúne material de apoyo para la cursada de **Ingeniería de Software 3**. Encontrarás resúmenes de los temas teóricos, prácticas resueltas y documentación complementaria que puede ser útil para estudiar, repasar o preparar parciales y finales.

> ⚠️ Este repositorio **no contiene código**. La materia es principalmente teórica/documental, por lo que todo el contenido es en forma de documentos y apuntes.

---

## 📁 Estructura del repositorio

```
IS3-UNLP/
│
├── resumenes/          # Resúmenes teóricos por unidad o tema
├── practicas/          # Prácticas resueltas
└── documentos/         # Material complementario, papers, apuntes varios
```

## 📂 Contenido

### 📝 Resúmenes
Resúmenes elaborados de los temas de la materia, organizados por unidad temática.

### ✏️ Prácticas resueltas
Resoluciones de ejercicios y prácticas de la cursada.

### 📄 Documentos
Material adicional que puede ser de utilidad: apuntes, papers, guías de estudio, etc.
